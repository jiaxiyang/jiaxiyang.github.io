radare2

  1. install
  1. show help

    usage

    1. r2 ./test
    2. r2 -A ./test analyse or r2 ./test; aaa
    3. ia show all info
    4. il libraries
    5. is show symbols
    6. iS show sections
    7. iX show used files
    8. ic show classes
    9. iE | grep FUNC show output functions

    ida

    1. 反编译工具集合
    2. IDA Pro 权威指南（第二版）.pdf
    3. Tab jump to pseudocode
    4. Esc jump to previous position
    5. Space text view and graph view 切换
    6. C-Enter jump to next position
    7. C-l jump by name + C-F filter
    8. C-p jump to function
    9. C-s jump to segment
    10. M-t +C-t find
    11. text view and graph view
    12. 窗口拖出来后，可以点击窗口灰色线条来 reattach, If you click on the small gray bar just below the titlebar, you can drag it back to the main window.
    13. view -> open subviews看各种窗口
    14. view -> toolbars -> debug 显示 debug 按钮
    15. 分割窗口，diassembly 和 pseudocode， 右键 synchronize with；可以高亮同步代码
    16. 新窗口会在鼠标停留的窗口打开

    debug

    1. 可以用 gdbserver 调试
    2. 可以实时看内存, bss, data, text
    3. 下方有gdb命令窗口， 可以exit来结束远程 gdbserver
    4. debugger -> detach from process
    5. debugger -> debugger windows 可以打开 stack trace(call stack), Locals 等调试窗口
    6. debugger -> using source level debugging 才可以在反编译的文件加断点

    修改二进制

    1. Edit -> Pach program -> change byte/assemble 修改二进制
    2. Edit -> Pach program -> Apply patches to input file 报错修改
    3. File -> Produce file -> Create DIF file 可以列出被修改的字节
    4. Compiler on line wandbox 可对比结果
    5. 汇编指令和二进制互转
    6. 汇编指令和二进制互转 1

    修改 PC 值

    1. 能临时跳到不同分支执行

    gdb

    1. b *main+12函数偏移来打断点
    2. disassemble 显示汇编
    3. p $pc显示 PC 值
    4. set var $pc=0x11111设置 PC 值

    plugins

    1. Virtuailor Virtuailor is an IDAPython tool that reconstructs vtables for C++ code written for intel architecture, both 32bit and 64bit code and AArch64 (New!).
    2. lighthouse

    ghidra

    1. 使用说明
    2. 需要先安装 jdk, 使用 cmd 打开， powershell 不行
    3. 开始界面有 debugger

    binary.ninja

    1. 破解
      • 需要替换安装路径下的动态库
      • 还不支持 debugger
    2. 可以远程调试
    3. 可以从不同 level 查看各个 section 内容
    4. 查看各种 level 代码
    5. 窗口划分，可同时显示不同级别代码
    6. 图形化做的不错
