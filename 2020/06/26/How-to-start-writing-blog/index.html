    Hexo basic comamnd

    1. Create a new post
      $ hexo new "My New Post"

    2. Run server
      $ hexo server

    3. Generate static files
      $ hexo generate

    4. Deploy to remote sites
      $ hexo clean && hexo deploy

    5. shortcuts
      alias hs='hexo clean && hexo g && hexo s'
      alias hd='hexo clean && hexo g && hexo d && git add . && git commit -m "update" && git push -f'
