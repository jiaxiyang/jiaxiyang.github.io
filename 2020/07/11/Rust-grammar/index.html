    1. Grammar

    Grammar

    1. indent with four spaces, not a tab

    2. Most lines of Rust code end with a semicolon(;)

    3. println!() ! means call macro, not a function

    4. Cargo is Rust’s build system and package manager.

    5. Cargo expects your source files to live inside the src directory. The top-level project directory is just for README files, license information, configuration files, and anything else not related to your code.

    6. cargo run Compile the code and then run the resulting executable all in one command.

    7. cargo check is much faster than cargo build

    8. cargo build --release compile the project with optimizations. Run faster but compile slower.

    9. The main function is the entry point into the program.

    10. The fn syntax declares a new function, the parentheses, (), indicate there are no parameters, and the curly bracket, {, starts the body of the function.

    11. let is used to create a variable. In Rust, variables are immutable by default.

       let foo = 5;  // immutable
 let mut bar = 5;  // mutable

    12. String::new() An associated function is implemented on a type, in this case String, rather than on a particular instance of a String. Some languages call this a static method. This new function creates a new, empty string. You’ll find a new function on many types, because it’s a common name for a function that makes a new value of some kind.
