keybind link

  1. [] 调上一个 func
  2. :e filename 打开 file
  3. za fold and unfold
  4. zM fold all
  5. zO unfold all
  6. C-^ 切换 buffer
  7. c-d 向下滚屏, 左手单手操作
  8. c-u 向上滚屏
  9. ZZ: 普通模式下保存并退出
  10. :e! 可以不保存文件并重新载入这个文件
  11. * 把当前的字段作为查找内容，找下一处。可以配合 n or N使用
  12. % 跳转到相配对的括号
  13. v % 选中括号里内容
  14. ^ 跳转至行第一个非空字符
  15. :set hlsearch 可以将查找的内容设置成高亮。:set nohlsearc 取消高亮或 /xxxx
  16. mm and `m : mark and jump back
  17. C-o 命令则跳到一个 “较老” 的地方 (提示： O 表示 older)。C-i 则跳到一个 “较新” 的地方.
  18. f 向后搜索, F 向前搜索, ; next
  19. H:将光标移动到当前可见页面的顶部（High）
  20. M:将光标移到当前可见页面的中间（Middle）
  21. L:将光标移动到当前可见页面的底部（Low）
  22. :buffers or :ls 列出可用的缓冲区
  23. :bn 下一个缓冲区。类似地，我们可以通过键入 b 数字或名称（在上面的第一列）或名称来切换到缓冲区。
  24. :bd 删除当前 buffer
  25. p and S-p 当前字符前后复制
  26. I go to start line and insert
  27. A go to end line and insert
