    Calc

    1. C-x * *
    2. calc-reset 重启
    3. D redo
    4. U or ctrl+/ undo
    5. Backspace calc-pop: can’t redo
    6. stack 计算方式, 可以直接输入+ - * /， 最近 stack 进行计算
    7. 计算 buf 均值
    8. C-x * q (quick-calc)
    9. step
      • put num in a buffer ctlr-x space select rectangle, remove other
      • select all buffer ctrl-x h
      • M-x: calc-grab-region
      • M-x: calc-vector-mean (u M)
    10. step
      • ctlr-x space select rectangle:(can slect string, such as: 10ms, 101ms NOTE: 后缀字符串必须一样)
      • M-x: calc-grab-rectangle
      • M-x: calc-vector-mean

    Rectange

    1. ctrl-x space + M-w + C-y 选择，复制，粘贴
    2. ctrl-x r
    3. ctrl-x r t insert
    4. ctrl-x r k kill
    5. ctrl-x r y paste
    6. ctrl-x r N insert a column of numbers

    ibuffer

    1. , b l
    2. h for help
    3. % n - Mark buffers by their name, using a regexp.
    4. U - Unmark all marked buffers.
    5. / n - Add a filter by buffer name.
    6. / / - Remove all filtering currently in effect.

    dired

    1. filter: search file and ivy-occur
    2. wdired-change-to-wdired-mode (可批量改文件名)Put a Dired buffer in Writable Dired (WDired) mode. C-c C-c wdired-finish-edit
    3. dired-hide-details-mode toggle dired details shortcut:(
    4. s sort by date
    5. ctrl+u s
    6. xah-dired-sort
    7. > next dir
    8. < previous dir
    9. f and b, j and k dired-find-file and dired-up-directory, next line and previous line
    10. + add directory
    11. C-x C-d recent dired

    speed up

    1. set windows repeat rate link
    2. profiler-start, profiler-report, profiler-stop to see cpu report
    3. next-line is slow: (setq auto-window-vscroll nil) link
    4. yascroll is slow; close
    5. recentf (set len 10)
    6. flycheck 应该很慢
    7. 如果卡了就完全从头编译一下

    Emacs Architecture

    macro

    1. kmacro-start-macro-or-insert-counter F3
    2. kmacro-end-or-call-macro F4
    3. call-last-kbd-macro C-x e e e e e
    4. F3 -> (F3) -> F4 counter
    5. C-u 222 F3 ... F3 ... F4 start counter from 222
    6. C-u 100 F4 repeat macro 100 times

    useful keys

    1. M-S-n 快速选中多行， 比M-; -> M-n更快
    2. repeat (global-set-key (kbd “C-x .”) ‘repeat)
    3. xterm-paste
    4. ibuffer -> % n -> “Occur” -> D remove all Occur buffer
    5. isserch: Typing M-s o in incremental search invokes isearch-occur
    6. occur 正则： yolox.Raw.running next-error
    7. clm/toggle-command-log-buffer C-c o
    8. in minibuffer: ivy-occur 默认绑定在C-c C-o上，它可以将当前的候选集合保存至 buffer 内并退出 minibuffer
    9. ivy-call 默认绑定在C-M-m上，它可以看做不退出 minibuffer 的 ivy-done 操作
    10. M-x M-o see ivy-mini-buffer-map; i: insert comand; d: go to defination;
    11. C-M-j ivy-immediate-done, when you call find-file to create a new file, but the desired name matches an existing file.
    12. 利用好 lsp
    (setq lsp-ui-doc-enable nil) ;; lsp关闭弹窗
(setq lsp-ui-sideline-mode nil) ;; lsp 关闭右侧错误提示
(setq lsp-signature-mode nil) ;; 防止M-n被占用
(setq lsp-enable-symbol-highlighting nil)) ;; lsp 不高亮

    编程操作

    1. symbol-overlay 高亮选中后可以用 t 切换 scope，然后 r 重命名。废弃： narrow => symbol-overlay-put => n or r跳转或替换 => fancy-widen
    2. symbol-overlay: 双击左键高亮或取消高亮， 中间键跳转到上一个， 右键跳转到下一个
    3. origami-toggle-all-nodes and origami-recursively-toggle-node, origami-show-only-node, origami-undo, origami-redo
    4. outline-show-all, outline-hide-other, outline-show-subtree
    5. bm-toggle and bm-next
    6. git-gutter: next, previous, revert
    7. view-mode: (help man 都适用)(单手，大拇指 space 无名指 Delete)
      • e quit and stay current buffer
      • space scroll page down
      • Delete scroll page up
      • (setq view-read-only t) read-only 打开 view-mode
