Chrome shortcuts

发表于 2021-05-21 更新于 2023-12-16 分类于 Tools ， Chrome 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

  1. 对视频右键两次，选择画中画模式，方便记笔记
  2. 番茄时钟画中画

shortcuts

  1. C-l或M-d: 地址栏
  2. M-Enter: open another tab when search: it’s very usefull
  3. C-k 或 C-e: 搜索
  4. Tab: 下一个可点击项
  5. M-left: 后退浏览历史
  6. M-right: 前进浏览历史
  7. C-r: 刷新网页
  8. Space: pgDn
  9. S-Space: pgUp
  10. C-t: 新标签
  11. C-n: 新窗口
  12. C-w: 关闭当前标签
  13. C-Tab: 切换标签
  14. C-S-t: 恢复关闭的标签
  15. C-h: history
  16. C-j: download
  17. C-1: 跳转到第一个标签
  18. C-9: 跳转到第一个标签
  19. C-Backspace: 删除一个单词
  20. S-Esc: 任务管理器
  21. C-S-b: 显示或隐藏书签栏
  22. C-f + C-g: 查找与跳转

自定义 shortcuts

  1. M-h：Select previous tab
  2. M-l：Select next tab
  3. M-k：Select last tab
  4. M-b: 后退浏览历史
  5. M-f: 前进浏览历史
  6. M-s: fullscreen
  7. M-3：new tab
  8. M-8：close tab
  9. M-w：Duplicate current tab

vimium 快捷键

  1. yy: copy URL to clipboard
  2. /: find
  3. j: Scroll down
  4. k: Scroll up
  5. h: Scroll left
  6. l: Scroll right
  7. gg: Scroll to the top of the page
  8. G: Scroll to the bottom of the page
